# Foodiz v3

Fondation frontend React + Vite pour Foodiz.

## Prérequis

- Node.js 18+
- npm

## Installation

```bash
npm install
```

## Lancement en développement

```bash
npm run dev
```

## Variables d'environnement

Copiez `.env.example` vers `.env` puis renseignez :

- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`

Aucune clé réelle n'est stockée dans ce repo.

## État de cette phase

Cette phase installe uniquement la fondation frontend :

- architecture React + Vite
- client Supabase centralisé
- routing principal par rôle
- layouts, guards et placeholders premium Foodiz
- design system de base

Les fonctionnalités produit avancées ne sont pas encore implémentées.
